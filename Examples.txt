##Example custom handler function in a frame using j$ library with event manager plugin

```
newset = {
	frames : [
		{
			elements : [div1, div2],
			actions : [
				function (e){
					j$(e).AddEvent({
						type : "click",
						func : function (v) {
							j$(v.toElement).RemoveAllEvents();
							console.log(v.toElement + " clicked");
						}
					});
				}
			],
			id : 1,
			data : []
		}	
	]
}
```

This would be automatically generated by
```
newset = jsa([div1, div2]).animate([{
	type : "custom", func : {
		function (e) {
			j$(e).AddEvent({
				type : "click",
				func : function (v) {
					j$(v.toElement).RemoveAllEvents();
					console.log(v.toElement + " clicked");
				}
			});
		}
	}
}]);
```

and then added to the loop with
```
jsa.insert.frameset(newset.frameset);
```

or, preferably,
```
newset.insertSets();
```
which will insert the main frameset and any secondary framesets into the main loop
